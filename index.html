Algunas preguntas que puede que alguien le guste saber sobre esta página web que hice.

¿Si me divertí? Por su puesto que si, para mí hacer esta página web fue algo que a principios no creí poder hacerla pero con las clases que se me dieron en el grupo del TeamRenaissance, fue muy fácil poder hacer todo esto, si quieren saber más de ellos que solo pude hablar un poco de cada profesor que ésta actualmente y uno que otro que estaba anteriormente.
¿Aprendí algo en el proceso de todo esta página web? Si, sinceramente si aprendí una que otra cosa, más que todo es tema de programación y esto para el futuro sea en un trabajo y diseñando páginas web para alguien me va a beneficiar, para mí esto es como un regalo y lo aprecio mucho.
¿cuesta hacer una página web así o algo que sé más llamativo sea de negocio, información sobre temas distintos o de algún otro tipo de página web? Hablando por mí no cuesta si tienes las ganas de aprender y de querer lograr hacer algo así o crear algo para variar y expresarte tus opiniones sobre ciertas cosas, a menos que quieras crear una página web para un negocio, ya esto requiere un poco más de ciertos programas como JavaScript, pero en lo demás solo seria un poco de tiempo que le inviertas.
